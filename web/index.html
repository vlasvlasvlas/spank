about spank instrument

This instrument mode is a minimal audiovisual layer built on top of spank: slap detection on Apple Silicon MacBooks that reacts with sound and visuals.

This version extends the original idea with a performance-oriented web interface, synth, configurable flash color, loops, and YAML-driven settings.

Current project: github.com/vlasvlasvlas/spank

Original project by taigrr: github.com/taigrr/spank
